# JaiminiDasha
A South Indian-style Jaimini Dasha research tool — single HTML file, no install. Paste JHora longitudes to compute Chara Karakas and 20+ dasha systems (Chara, Narayana, Sthira, Trikona, Drig, Karaka dashas &amp; more) with mahadasha/antardasha breakdowns, multi-chart support, and printable PDF reports. 100% client-side &amp; private.
